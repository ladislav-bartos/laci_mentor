# EVENTS.md — Networking Events Tracker (living doc)

Purpose: track concrete, recurring networking events/orgs worth attending,
their cadence, and why they fit the Ideal Contact Profile (see `FACTS.md`)
— so picking "2-3 candidate events" each sprint is a lookup, not a fresh
search every time. Complements `SOURCES.md` (online contact-sourcing
channels) — this file is about events to physically/virtually attend.

## How to use this file
- Check the "Where to check dates" URL for each candidate before
  committing — cadence here is typical pattern, not a live calendar.
- Cross-check against `SPRINT.md`'s scheduling-constraints section
  (family/wife's schedule) before booking anything, per CLAUDE.md.
- Weekends are never in scope — evening events only work on nights without
  a family commitment; day events need to fit inside the 2-3 hr/weekday
  work capacity.
- "ICP fit" flags whether the typical audience actually includes his
  target titles (CEO/COO/CPO, Head/VP Product, Country Manager Japan,
  CHRO/Head of Talent) — events skewed to early-stage founders or IC-level
  peers are noted as lower priority, not removed, since peer intel /
  volume still has some value.
