# SOURCES.md — Contact Sourcing Research (living doc)

Purpose: candidate channels for finding new contacts that fit the Ideal
Contact Profile (growth-stage-to-enterprise companies expanding/scaling in
Japan/APAC — see `FACTS.md`), without paying for LinkedIn Sales Navigator
yet (deferred until warm outreach proves out — see `LEARNINGS.md`
2026-07-16).

Updated two ways:
- Manually, whenever a source is tried
- Via `/research-sources` — an on-demand command that goes and finds more
  candidate sources, using a different angle each run so we don't just
  re-search the same "top VC firms in Japan" query repeatedly

## How to use this file
- **Sources table**: one row per channel. Fill in "My notes" and "Verified"
  after you've actually tried a source — that feedback is what lets
  `/research-sources` learn what's working and steer future runs away from
  dead ends.
- **Research Log**: append-only history of what `/research-sources` tried,
  so each run builds on the last instead of repeating it.
